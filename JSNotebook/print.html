My JavaScript Notebook: "Print()" and "Println()" functions!

What I learned today was how to coherently use the "print()" and "println()" functions! The function "print()" lets the person typing type their information, and the person coding uses this to make information print out on the same line. On the other hand, the function "println()" is used when the person typing type his/her information, and the person coding wants what prints out (which is the users information) to be on a new line. Those are the distinct differences between the two functions "print()" and "println()"!
